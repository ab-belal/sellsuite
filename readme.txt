A modular WooCommerce extension that enhances store management with customer loyalty points, product management, and customer dashboard.

== Description ==

SellSuite is a comprehensive WooCommerce extension that adds powerful features to your online store:

* **Customer Loyalty & Points System** - Reward customers with points for purchases
* **Product Management Extensions** - Enhanced product management capabilities
* **Customer Management Dashboard** - Comprehensive customer insights and management

== Features ==

* Award points automatically on order completion
* Display customer points on account page
* Modern React-based admin dashboard
* REST API for extensibility
* Fully integrated with WooCommerce

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sellsuite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Navigate to WooCommerce > SellSuite to configure the plugin

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, SellSuite requires WooCommerce to be installed and active.

= How are points calculated? =

Points are calculated based on the order total. You can configure the points-per-dollar ratio in the settings.
